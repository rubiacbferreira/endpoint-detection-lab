# osquery Configuration

This directory contains configuration files and query packs used for detection engineering.

---

## Packs

Located in:

packs/

These files define scheduled queries that can be executed automatically by osquery.

---

## Purpose

The goal of these packs is to:

- automate detection logic
- collect telemetry from endpoints
- support MITRE ATT&CK aligned detections

---

## Status

Currently under development.

Future versions of the lab will integrate these packs directly into the osquery daemon configuration running on the endpoints.
